  • What are the different phases of React components lifecycle?
    • In ReactJS, every component creation process involves various lifecycle methods. These lifecycle methods are termed as component's lifecycle. These lifecycle methods are not very complicated and called at various points during a component's life. The lifecycle of the component is divided into four phases. They are: 1. Initial Phase 2. Mounting Phase 3. Updating Phase 4. Unmounting Phase Each phase contains some lifecycle methods that are specific to the particular phase
